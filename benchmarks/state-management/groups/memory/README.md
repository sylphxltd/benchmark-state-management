## Technical Details

**Description**: Memory tests evaluate object creation and garbage collection.

**Test Scales**: `Light`, `Moderate`, `Heavy`

**Focus Areas**: - Memory allocation
- GC pressure
- Object lifecycle

**Library Interface**: All libraries implement a standardized interface with these methods:
 `get count`, `increment`, `setNested`, `addUser`, etc.

**Measurement**: Each test runs multiple iterations and reports:
- Operations per second (ops/sec)
- Mean execution time
- 99th percentile (p99)
- Statistical variance
